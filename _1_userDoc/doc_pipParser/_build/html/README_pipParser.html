Description¶

Ce petit programme permet de créer le fichier ‘requierement’ à partir de la commande ‘pip3 freeze’. Ce fichier est utile pour l’installation des paquet sous python.

Une première variante du fichier ‘nettoyé’ des numéros de versions peut être généré.

Une seconde version ‘customisé’ peut être généré pour n’avoir que les paquet présent dans un dossier donné.
